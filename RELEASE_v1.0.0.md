# 豆荚 v1.0.0

首个可发布版本。

## 功能

- 顶部精简
  - 隐藏同城、商城、团购、热点、经验、精选
  - 隐藏“返回推荐页”按钮

- 底部精简
  - 隐藏朋友
  - 隐藏发布/拍摄

- 首页透明度
  - 右侧操作区透明度
  - 左下文案透明度
  - 右下音乐区透明度
  - 状态栏透明
  - 底部栏透明

- 视频过滤
  - 过滤关键词：广告、推广
  - 过滤低赞视频：点赞数低于 100
  - 过滤直播流

## 兼容说明

- 模块框架：LSPosed / libxposed api 101
- 静态作用域：`com.ss.android.ugc.aweme`
- 当前仓库为固定配置版本，不依赖独立前端设置页

- 不提供目标应用资源或反编译产物
- 仅供学习和研究使用
